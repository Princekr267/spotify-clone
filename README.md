### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge, etc.)

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Princekr267/spotify-clone.git
   ```

2. Navigate to the project directory
   ```bash
   cd spotify-clone
   ```

3. Open the `spotifyClone.html` file in your browser
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser

That's it! No server or build process required.
